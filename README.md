# Harp Helper Script

Simple Hypixel SkyBlock Harp helper using MineScript.

Detects quartz blocks in the GUI and clicks them with minimal random delay.

All Songs 95+% Possible (Ping dependant)

## Requirements

- MineScript
- Minescript-Plus

Minescript-Plus is required for inventory interaction:
https://github.com/R4z0rX/minescript-scripts/tree/main/Minescript-Plus
